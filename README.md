# Red_neuronal_Digitos_Manuscritos
Este proyecto consiste en la construcción, entrenamiento y evaluación de una red neuronal para la clasificación de imágenes de dígitos escritos a mano, utilizando el dataset MNIST.

El objetivo principal fue desarrollar un modelo de Deep Learning capaz de reconocer correctamente números del 0 al 9 a partir de imágenes en escala de grises.

Este trabajo fue desarrollado como parte de mi formación en el área de Data Science, aplicando conocimientos de Python, redes neuronales, procesamiento de datos e interpretación de resultados.

____________________________________________________________________________________________________________________________________________________________

##bjetivo

Desarrollar un modelo de red neuronal artificial que permita clasificar imágenes de dígitos manuscritos, evaluando su rendimiento mediante métricas como la precisión y la pérdida del modelo.

___________________________________________________________________________________________________________________________________________________________

##Dataset utilizado

Se utilizó el dataset MNIST, una base de datos ampliamente utilizada en proyectos de Machine Learning y Deep Learning.
El conjunto de datos contiene imágenes de dígitos manuscritos del 0 al 9, divididas en datos de entrenamiento y prueba.
Cada imagen tiene un tamaño de 28x28 píxeles y se encuentra representada en escala de grises.

##Etapas del proyecto

1. Importación de librerías

Se utilizaron librerías de Python para el manejo de datos, visualización de imágenes y construcción del modelo de red neuronal.

_______________________________________________________________________________________________________________________________________________________

2. Carga del dataset

Se cargó el dataset MNIST, separando los datos en conjuntos de entrenamiento y prueba.

_____________________________________________________________________________________________________________________________________________________

3. Exploración de datos

Se visualizaron algunas imágenes del dataset para comprender la estructura de los datos y las etiquetas asociadas.

___________________________________________________________________________________________________________________________________________________

4. Preprocesamiento

Las imágenes fueron normalizadas para mejorar el rendimiento del modelo. Además, las etiquetas fueron transformadas mediante One-Hot Encoding para trabajar con clasificación multiclase.

________________________________________________________________________________________________________________________________________________________

5. Construcción del modelo

Se implementó una red neuronal utilizando TensorFlow y Keras, compuesta por capas densas para procesar la información de las imágenes y clasificar los dígitos.

________________________________________________________________________________________________________________________________________________________

6. Entrenamiento

El modelo fue entrenado utilizando los datos de entrenamiento, ajustando los pesos de la red neuronal durante varias épocas.

_______________________________________________________________________________________________________________________________________________________

7. Evaluación

Finalmente, el modelo fue evaluado utilizando el conjunto de prueba para medir su capacidad de generalización frente a datos no vistos durante el entrenamiento.

________________________________________________________________________________________________________________________________________________________

##Resultados obtenidos

El modelo logró un buen desempeño en la clasificación de dígitos manuscritos, alcanzando una precisión aproximada de 98% en los datos de prueba.

Este resultado demuestra que una red neuronal puede aprender patrones visuales relevantes y clasificar correctamente imágenes de números escritos a mano.
